# Chat

Talk to your agents here. Each column is a separate conversation with an agent.

### Columns
- Click **+ Add Column** to open a new conversation
- Each column targets one agent — pick from the dropdown
- Close columns with the X button in the header

### Mentions
- Type `@` in the input to mention another agent by name
- The mentioned agent receives context about the conversation

### Delegate
- Click the **Delegate** button to hand off a task to a sub-agent
- The master agent creates a job on the shared board and a specialised agent picks it up

### Voice Input
- Click the microphone button for voice-to-text input
- Speak naturally — the transcription is sent as your message

### Page Context
- Click **Read Page** to grab the current tab's content
- The agent can then answer questions about the page

### Keyboard Shortcuts
- `Enter` — Send message
- `Shift+Enter` — New line
- `Ctrl+Shift+C` — Open CHAOS
- `Ctrl+Shift+U` — Voice input
