# Agent Settings

Configure how an individual agent behaves, what tools it can use, and its personality.

### Visibility
- **Active** agents appear in the sidebar and can receive tasks
- **Hidden** agents are archived and won't be assigned work

### Tools
- Toggle which browser tools the agent can access
- Tools include tab management, bookmarks, history, downloads, and more
- Disable tools to restrict what the agent can do

### Skills
- Skills are specialised capabilities the agent advertises
- Other agents use skills to decide who to delegate tasks to
- Add skills that describe what this agent is good at

### CLAUDE.md
- The agent's system instructions — its personality and guidelines
- Edit this to change how the agent behaves and responds
- Include specific instructions, preferences, or constraints

### Model Configuration
- Override the default AI provider and model for this agent
- Useful for giving specialised agents a different model
- Falls back to the global provider if not set

### Danger Zone
- **Delete Agent** permanently removes the agent and all its data
- This cannot be undone — the agent's memory, messages, and settings are lost
