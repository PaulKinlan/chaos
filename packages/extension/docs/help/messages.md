# Messages

Messages show inter-agent communication — what agents send to each other during task delegation and collaboration.

### Sent and Received
- **Sent** messages are ones this agent dispatched to other agents
- **Received** messages are ones sent to this agent
- Use the filter dropdown to show only sent or received

### How Delegation Works
- When an agent delegates a task, it sends a message to the sub-agent
- The sub-agent receives the message with instructions and context
- Replies flow back as the sub-agent reports progress or results

### Searching
- Use the search box to find messages by content
- Helpful for tracking down specific task communications
