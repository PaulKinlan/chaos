# Artifacts

Artifacts are files that agents create and share — HTML pages, documents, data, images, and more.

### What Gets Created
- Agents publish artifacts when they build something tangible
- Common types: HTML pages, reports, summaries, generated content

### Viewing Artifacts
- Click an artifact card to preview it
- HTML artifacts render in a sandboxed iframe

### Publishing
- Agents automatically publish artifacts during task execution
- You can also ask an agent: "Create a page about..."

### Copying and Downloading
- Use the copy button to grab artifact content
- Download artifacts to save them locally

### Deleting
- Click the delete button on an artifact card
- This removes it from the shared artifact store
