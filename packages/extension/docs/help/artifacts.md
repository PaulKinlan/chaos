# Artifacts

Files and content that agents create and share -- HTML pages, documents, data, images, and more.

## What This View Does

Artifacts are the tangible outputs of agent work. When an agent builds a page, generates a report, or creates any file, it publishes it as an artifact. This view lets you browse, preview, and manage everything your agents have produced.

## What Gets Created

- **HTML pages** -- interactive dashboards, reports, visualizations
- **Documents** -- summaries, analyses, research notes
- **Data files** -- CSV exports, JSON data, structured output
- **Images** -- generated graphics, charts, screenshots
- Agents decide what to publish based on the task -- ask them to "create a page" or "build a report"

## Viewing Artifacts

- Click an artifact card to preview it
- HTML artifacts render in a sandboxed iframe so you can interact with them
- Text artifacts display with syntax highlighting
- Each artifact shows which agent created it and when

## Pinning

- Pin important artifacts to keep them at the top of the list
- Pinned artifacts also appear on the Dashboard for quick access
- Unpin when you no longer need quick access

## Copying and Downloading

- Use the **copy** button to grab artifact content to your clipboard
- Click **download** to save an artifact as a local file
- Useful for sharing agent output outside of CHAOS

## Deleting

- Click the delete button on an artifact card to remove it
- Deleted artifacts cannot be recovered
- Agents can always create new artifacts if needed

## Tips

- Ask agents to "create an artifact" or "publish a page" when you want shareable output
- Pin dashboards or reports you check regularly
- Download artifacts you want to keep long-term -- the artifact store is not a permanent archive
- HTML artifacts can include JavaScript, CSS, and interactive elements
- Use artifacts as a way to get structured output from agents instead of just chat text
