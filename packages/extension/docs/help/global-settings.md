# Global Settings

Configure providers, API keys, appearance, and permissions that apply to all agents.

### Providers
- Select your active AI provider (Anthropic, Google, OpenAI, OpenRouter, Ollama)
- Enter your API key for the selected provider
- Choose which model to use from the provider's available models

### API Keys
- Each provider needs its own API key (except Ollama, which runs locally)
- Keys are stored locally in your browser — never sent anywhere except the provider
- Test your key by sending a message in chat

### Theme
- Switch between light, dark, and system theme
- The system option follows your OS preference

### Permissions
- Control what agents can do by default
- **Auto-approve** lets agents act without asking
- **Ask every time** requires your confirmation for each action
- **Deny** blocks the action entirely

### Archived Agents
- View and restore previously archived agents
- Archived agents keep their data but don't appear in the active sidebar
